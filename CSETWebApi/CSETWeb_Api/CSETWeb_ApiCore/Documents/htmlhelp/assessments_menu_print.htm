Assessments Menu

 
The Assessments menu allows quick access to the assessment questions and categories. The figure below shows the Assessments menu navigation.
 
NOTE: The Assessment menu will look different depending on what is selected on the Assessment Configuration screen. To learn more about navigation, see CSET Navigation.
Requirements mode navigation will differ in that it shows standards at the top level and then categories nested underneath them.
 
Figure: Assessment button/menu
 

Assessments tab

 
Clicking the Assessments tab will display the Questions screen displayed after the Preparation process.
 
See the Assessment Section for more information about the Questions screen.

Navigation toggle

Use the Navigation toggle to open and close the Navigation Menu.

Assessment menu

 
The Assessment menu shows a list of all question categories awaiting completion for the assessment. The user can quickly navigate to a specific category by clicking the desired menu item.
